# Doompedia Android

Native Android client (Kotlin + Jetpack Compose) for the offline-first summary-card experience.

## Implemented in this scaffold
- Room schema for 1M-ready article card metadata.
- Asset bootstrapper with normalized titles and aliases.
- Local pack install classes: `PackInstaller`, `DeltaApplier`, and resumable `ShardDownloader`.
- Background updates using `PackUpdateWorker` (periodic checks + delta/full apply).
- Deterministic ranking via shared `ranking-config.v1.json`.
- Title search: exact + prefix + alias + typo distance <= 1.
- Personalization controls: `OFF`, `LOW`, `MEDIUM`.
- Light/dark/system theme modes.
- Card action opens Wikipedia universal URL.

## Build
- JDK 17
- Android SDK 35

Open `android-app/` in Android Studio and sync Gradle.

## Notes
- `seed_en_cards.json` is demo content only. Production flow uses shard imports generated by `data-pipeline`.
- Search is title-only in MVP for predictable storage and latency.
- Set a manifest URL in Settings to enable manual and periodic pack updates.
