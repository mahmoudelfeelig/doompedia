# Delivery Roadmap

## Milestone A (implemented in this scaffold)
- Shared contracts for schema/manifest/ranking.
- Android and iOS native scaffolds with:
  - local DB,
  - feed UI,
  - title search,
  - personalization controls,
  - deep-link opening.
- Pipeline utilities for shard and delta generation.

## Milestone B
- Production-grade dump ingestion hardening (baseline XML extractor is implemented).
- Download manager with resumable shard network transfers.
- Background periodic update checks.
- Static-host publishing/deployment workflow for pack artifacts.
- Full accessibility and performance verification on device matrix.

## Milestone C
- Additional language packs.
- Advanced topic curation controls.
- Optional exported reading lists (local file).
